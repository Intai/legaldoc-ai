   Each section must fully address the topics described in the outline. Do not leave placeholders or incomplete provisions.

   Incorporate the specific details, preferences, and requirements provided by the user. Where the user context conflicts with the reference material, prioritize the user context.

   Draft provisions that are fair and balanced unless the user context specifies otherwise. Avoid one-sided terms that could be challenged for unconscionability.
